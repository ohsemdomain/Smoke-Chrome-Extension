# **App Name**: Smoke Break Tracker

## Core Features:

- Log Cigarette: A single-click button to log the current time as a time when the user smoked a cigarette.
- Daily Count Display: Display the number of cigarettes smoked so far today.
- Last Smoked Timer: A timer displaying how long it has been since the last cigarette was logged.

## Style Guidelines:

- Primary color: Deep green (#386641) to represent health and nature, providing a calm feel.
- Background color: Muted light gray (#E0E0E0) to provide a clean and minimalist backdrop.
- Accent color: A slightly brighter, but desaturated green (#748A79) for highlighting key elements.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and readable interface.
- Use simple, minimalist icons for the button and any other UI elements.
- A clean, spacious layout focusing on clarity and ease of use with well-placed key information.
- Subtle animations, such as a brief pulse, upon logging a cigarette to give visual feedback.
